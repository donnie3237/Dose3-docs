&nbsp;&nbsp;&nbsp;to use dose cli you need to run this command in your therminal then get enjoy

```shell
npm i -g dose3@latest
```
### NodeJS

&nbsp;&nbsp;&nbsp; this cli based on Nodejs npm package and written in Javascript code in this case we need to install nodejs first <a href="https://nodejs.org/en/" target="blank">click to download</a>

### Git
&nbsp;&nbsp;&nbsp; in this process we use git repository to clone project to you if you want to use it you should install git <a href="https://git-scm.com/" target="blank">click to download</a>

### VScode
&nbsp;&nbsp;&nbsp; we can integrate with vscode to make your life faster to open project with vscode you should download vscode to start project <a href="https://code.visualstudio.com/" target="blank">click to download</a>
